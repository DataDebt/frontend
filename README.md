# ⚡️ DataDebt

**DataDebt** Data Deb is the web platform that guides organizations through a structured process to analize the internal data deb.


&nbsp;

## ⚙️ Installation and Setup

Open your terminal and run the following commands to install and run the application.

**Clone the repository**
```bash
git clone https://github.com/DataDebt/frontend.git
cd frontend
```

**Install the required dependencies**
```bash
npm install
```

**Run the application in a development environment**
```bash
npm run dev
```
*You can access the app in your browser at [http://localhost:9002](http://localhost:9002)*

**Build for production**
```bash
npm run build
```

**Run in a production environment**
```bash
npm start
```
*By default, you can access the app in your browser at [http://localhost:3000](http://localhost:3000)*
